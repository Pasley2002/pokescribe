# ✍️ PokeScribe
[**English**](#english) | [**Español**](#español)

---

<a name="english"></a>
## 🇺🇸 English
**PokeScribe** is a web tool optimized for the **Cobblemon** community. It automatically converts Pokémon sets from **Showdown** into `/pokegive` and `/pokeedit` commands.

### ✨ Main Features:
* **Full Regional Support:** Alola, Galar, Hisui, Paldea, and the custom **Kazeran** (Orizon) regional form.
* **EV Validation:** Visual alert if the set exceeds the 510 regulation points.
* **Format Intelligence:** Detects moves even from Discord (using bullets • or asterisks *).
* **ID Formatting:** - Abilities/Moves: Joined format (`ironfist`).
  - Items: snake_case format (`heavy_duty_boots`).
* **Shiny Support:** Automatic detection of shiny sets ✨.
* **Multilingual UI:** Support for both English and Spanish.

### 🚀 How to use:
1. Paste your Showdown set into the text box.
2. Click on **Generate Team**.
3. Copy the commands and use them in Minecraft.

<a name="español"></a>
## 🇲🇽 Español
**PokeScribe** es una herramienta web optimizada para la comunidad de **Cobblemon**. Convierte sets de Pokémon de **Showdown** a comandos `/pokegive` y `/pokeedit` de forma automática.

### ✨ Características principales:
* **Soporte regional completo:** Alola, Galar, Hisui, Paldea y la forma regional **Kazeran** (Orizon).
* **Validación de EVs:** Alerta visual si el set excede los 510 puntos reglamentarios.
* **Inteligencia de Formato:** Detecta movimientos incluso si vienen de Discord (usando puntos • o asteriscos *).
* **ID Formatting:** - Habilidades/Movimientos: Formato pegado (`ironfist`).
  - Objetos: Formato snake_case (`heavy_duty_boots`).
* **Soporte Shiny:** Detección automática de sets variocolor ✨.
* **Interfaz Bilingüe:** Soporte para Inglés y Español.

### 🚀 Cómo usar:
1. Pega tu set de Showdown en el cuadro de texto.
2. Haz clic en **Generar Equipo**.
3. Copia los comandos y úsalos en Minecraft.
