# ✍️ PokeScribe
**PokeScribe** es una herramienta web optimizada para la comunidad de **Cobblemon**. 
Convierte sets de Pokémon de **Showdown** a comandos `/pokegive` y `/pokeedit` de forma automática.

### ✨ Características principales:
* **Soporte regional completo:** Alola, Galar, Hisui, Paldea y la forma regional **Kazeran** (Orizon).
* **Validación de EVs:** Alerta visual si el set excede los 510 puntos reglamentarios.
* **Inteligencia de Formato:** Detecta movimientos incluso si vienen de Discord (usando puntos • o asteriscos *).
* **ID Formatting:** - Habilidades/Movimientos: Formato pegado (`ironfist`).
  - Objetos: Formato snake_case (`heavy_duty_boots`).
* **Soporte Shiny:** Detección automática de sets variocolor ✨.

### 🚀 Cómo usar:
1. Pega tu set de Showdown en el cuadro de texto.
2. Haz clic en **Generar Equipo**.
3. Copia los comandos y úsalos en Minecraft.
